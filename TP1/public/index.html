Cocktail Database

Cocktails Endpoints

  • GET /api/cocktails - Trae todos los cocktails de la base de datos.
  • POST /api/cocktails - Agrega un nuevo cocktail a la base de datos. Ejemplo de estructura: 
                            {
                            "iba": true,
                            "name": "Horse's Neck",
                            "colors": "#D9A031",
                            "glass": "highball",
                            "category": "Longdrink",
                            "ingredients": [
                              { "unit": "cl", "amount": 4, "ingredient": "Cognac" },
                              { "unit": "cl", "amount": 12, "ingredient": "Ginger Ale" },
                              { "special": "Dash of Angostura bitters (optional)" }
                            ],
                            "garnish": "Lemon twist",
                            "preparation": "Build into highball glass with ice cubes. Stir gently. If required, add dashes of Angostura bitters."
                          }
  • GET /api/cocktails/:id - Trae un cocktail por ID.
  • PUT /api/cocktails/:id - Modifica un cocktail en la base de datos por ID. Ejemplo de estructura (no se necesita rellenar todos los campos a la vez) : 
                            {
                            "iba": true,
                            "name": "Horse's Neck",
                            "colors": "#D9A031",
                            "glass": "highball",
                            "category": "Longdrink",
                            "ingredients": [
                              { "unit": "cl", "amount": 4, "ingredient": "Cognac" },
                              { "unit": "cl", "amount": 12, "ingredient": "Ginger Ale" },
                              { "special": "Dash of Angostura bitters (optional)" }
                            ],
                            "garnish": "Lemon twist",
                            "preparation": "Build into highball glass with ice cubes. Stir gently. If required, add dashes of Angostura bitters."
                          }
  • DELETE /api/cocktails/:id - Borra un cocktail de la base de datos por ID.

Users Endpoints

  • /api/cocktails - Trae todos los cocktails de la base de datos.
  • /api/users - Trae todos los users de la base de datos.
